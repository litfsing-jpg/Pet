# Интерактивность платформы услуг

## Основные взаимодействия

### 1. Карточки услуг
- **Наведение**: 3D-трансформация, увеличение, появление тени
- **Клик**: Открытие модального окна с детальной информацией
- **Фильтрация**: Кнопки категорий для фильтрации услуг

### 2. Модальные окна
- **Просмотр деталей**: Описание услуги, цены, сроки
- **Галерея примеров**: Слайдер с примерами работ
- **Форма заказа**: Быстрая связь через Telegram

### 3. Интерактивные элементы
- **Кнопка "Заказать"**: Открывает чат в Telegram
- **Просмотр примеров**: Галерея с возможностью увеличения
- **FAQ секция**: Аккордеон с вопросами и ответами

### 4. Навигация
- **Плавная прокрутка** к секциям
- **Активное состояние** пунктов меню
- **Мобильное меню** для малых экранов

## Пользовательский путь
1. Пользователь заходит на главную страницу
2. Ознакамливается с услугами через карточки
3. Нажимает на интересующую услугу
4. Просматривает примеры работ в модальном окне
5. Нажимает "Заказать" и переходит в Telegram для связи

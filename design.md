# Дизайн-концепция платформы цифровых услуг

## Общий стиль
- **Темная тема** с элегантным и современным дизайном
- **Цветовая палитра**: Темно-серый фон (#0a0a0a), акцентные бирюзовые (#00d4aa) и фиолетовые (#8b5cf6) оттенки
- **Типографика**: Чистые sans-serif шрифты, выразительные заголовки
- **Визуальный язык**: Минималистичный с фокусом на контент и функциональность

## Визуальные эффекты
- **Анимированные фоновые элементы** с помощью shader-park
- **Плавные переходы** между секциями
- **Наведение на карточки** с 3D-эффектами
- **Параллакс-эффекты** при скролле
- **Анимированные иконки** и интерактивные элементы

## Компоненты
- **Герой-секция** с динамическим фоном и привлекающим заголовком
- **Карточки услуг** с hover-эффектами и примерами работ
- **Модальные окна** для детальной информации
- **Форма обратной связи** с интеграцией Telegram
- **Галерея примеров** работ для каждой услуги

## Интерактивность
- **Плавная навигация** по странице
- **Фильтрация услуг** по категориям
- **Просмотр примеров** работ в модальном окне
- **Быстрая связь** через Telegram бот
